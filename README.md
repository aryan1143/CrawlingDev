# CrawlingDev

## Project Description

CrawlingDev is a platform where users can list their projects and other developers can rate them and give suggestions for improvement.

## Tech Stack

### Frontend

- React
- Vite
- Redux Toolkit
- React Redux
- React Router DOM
- Tailwind CSS
- React Hook Form
- React Hot Toast
- Lucide React

### Backend

- Node.js
- Express
- JWT (jsonwebtoken)
- bcrypt
- cookie-parser
- CORS
- dotenv
- nodemon

## Project Structure

- client: Frontend application
- server: Backend API and server logic
- docs: Project documentation
